# Machine Learning

## Overview
Simple Machine Learning library written in C++

Provides Neural Network and basic linear algebra libraries

## Features
- Backpropagation
- Multiple activation functions (Sigmiod, ReLU, Tanh) 
- Stochastic Gradient Descent
- Randomization
